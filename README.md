### 🍲 KingPao — Online Ordering Platform

> A full-stack food delivery platform built from scratch for a Filipino siopao restaurant chain (est. 1985). Think a custom-branded GrabFood — but engineered for a single brand.

<table>
<tr>
<td width="50%">

**🚀 What It Does**
- Online menu browsing, cart & checkout
- Real-time order tracking
- Branch-aware delivery zones with geo-fencing
- Digital wallet (KingWallet) with auto-refunds
- AI chatbot (Gemini-powered)
- PWA — installable on mobile

</td>
<td width="50%">

**📊 Technical Highlights**
- **40+ database tables** — fully normalized MySQL schema
- **50+ API endpoints** — PHP 8.x backend
- **EMA algorithm** — self-learning delivery fee estimation
- **7-tab analytics dashboard** — Chart.js + Leaflet maps
- **Multi-role system** — Customer, Reseller, Branch Admin, Super Admin
- **No frameworks** — built with vanilla HTML/CSS/JS/PHP

</td>
</tr>
</table>

<details>
<summary>📐 <strong>Architecture & Tech Stack</strong></summary>
<br/>

| Layer | Technology |
|-------|-----------|
| **Frontend** | Vanilla HTML / CSS / JavaScript (ES6+) |
| **Backend** | PHP 8.x on Apache (XAMPP) |
| **Database** | MySQL 8.x — 40+ tables, views, migrations |
| **Maps & Geo** | Leaflet.js · Leaflet Routing Machine · Turf.js · OSRM API |
| **Payments** | Xendit · PayMongo (Philippine gateways) |
| **AI** | Gemini API (chatbot) |
| **Auth** | Session-based · OTP via Twilio SMS · CAPTCHA |
| **Charts** | Chart.js |
| **PWA** | Service Worker · Web App Manifest |

</details>
